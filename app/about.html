About ImageShrinker

ImageShrinker is a simple tool to shrink images. Although there are countless number of websites that do just this, having an offline tool for the purpose of shrinking images makes things a bit easier. Also, the images that you resize won't be uploaded on a server so this app is a much faster alternative.

For Developers

If you are a developer, you can unlock a hidden developer's menu that lets you toggle the developer's tools and do other cool stuff. To do this, get the project's code from here and in main.js, set the development environment to development by setting process.env.NODE_ENV to development.
